## @tanstack/react-virtual

- **License:** MIT
- **Source:** https://github.com/TanStack/virtual
- **Version:** ^3.10.9
- **Use:** Row virtualization in DataTable — renders only the rows visible
  in the viewport to handle 10k+ records without DOM bloat.
- **Why chosen:** Lightweight (no side-effects, framework-agnostic core),
  well-maintained, idiomatic React API. Avoids the complexity of a full
  AG Grid / react-window setup for this kit's scope.
